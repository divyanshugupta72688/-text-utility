# 📝 Web Text App - Text Utility Tool

A full-stack Text Utility Web Application built with React.js frontend 
and Node.js backend. Helps users manipulate, analyze and transform 
text in real-time.

## 🛠️ Tech Stack

Frontend:
- React.js (Create React App)
- HTML5, CSS3
- JavaScript ES6+

Backend:
- Node.js
- Express.js

---

## ✅ Features
- Convert text to UPPERCASE or lowercase
- Count words and characters in real-time
- Remove extra spaces from text
- Copy text to clipboard
- Clear text instantly
- Responsive design works on all devices
- Light and Dark mode support

## 🚀 How to Run Locally

Prerequisites - Make sure you have installed:
- Node.js v14 or above from https://nodejs.org
- npm comes with Node.js automatically

Step 1 - Clone the Repository
git clone https://github.com/divyanshugupta72688/text-utility.git
cd text-utility

Step 2 - Run the Backend
cd backend
npm install
node server.js
Backend runs at http://localhost:5000

Step 3 - Run the Frontend in a New Terminal
cd frontend
npm install
npm start
Frontend runs at http://localhost:3000

Step 4 - Open in Browser
http://localhost:3000

## 🌐 Deployment

Frontend - Deploy on Netlify FREE
1. Go to https://netlify.com
2. Drag and drop the frontend/build folder
3. Get instant live URL

Backend - Deploy on Render FREE
1. Go to https://render.com
2. Connect your GitHub repo
3. Set root directory as backend
4. Start command: node server.js
